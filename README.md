# לוח תקשורת — אפליקציית AAC עברית

אפליקציית **תקשורת תומכת וחליפית (AAC)** עברית-ראשונה לילדים עם קשיי תקשורת (דגש אוטיזם), לקלינאי תקשורת והורים. מסחרי · אנדרואיד + Web תחילה, iPad בהמשך.

> **סטטוס:** אפיון (PRD) הושלם. הקוד טרם נכתב — ראו `EXECUTION-PROMPT.md` לבניית ה-MVP.

## מבנה ה-repo
```
.
├── PRD/
│   ├── PRD-he.md          # מסמך האפיון המלא (עברית, ראשי) — מקור האמת
│   └── PRD-en.md          # גרסה אנגלית מקבילה
├── HANDOFF.md             # סקירת פרויקט — לקרוא בתחילת כל סשן
├── EXECUTION-PROMPT.md    # פרומפט בנייה לסשן חדש (תהליך + תתי-סוכנים + בדיקות)
├── GEMINI.docx            # 4 מסמכי המחקר המקוריים (בשורש ה-repo)
├── chat gpt.docx
├── אופוס.docx
├── חוקר 1.docx
└── README.md
```

## מאיפה להתחיל
1. קראו את `HANDOFF.md` (2 דקות) לסקירה.
2. קראו את `PRD/PRD-he.md` לאפיון המלא.
3. לבנייה: פתחו סשן חדש והדביקו את תוכן `EXECUTION-PROMPT.md`.

## עיקרי המוצר
- עברית עמוקה: RTL, ניקוד אוטומטי, מורפולוגיה, TTS עברי איכותי.
- שני מצבי UX: הורה פשוט / קלינאי עוצמתי.
- Offline-first; Builder מודרני עם עריכה קבוצתית; מדידה קלינית מובנית.
- MVP: 15 פיצ'רי חובה (ראו PRD §11).
