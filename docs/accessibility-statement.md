# הצהרת נגישות — Co_Board

**רמת התאמה:** התאמה חלקית (Partially Conformant) לתקן הישראלי **ת"י 5568**,
המאמץ את הנחיות **WCAG 2.0 רמה AA**.

**תאריך עדכון אחרון:** 2025-01 (v1)
**גרסת מסמך:** 1.0

## 1. מחויבות לנגישות

‏Co_Board היא אפליקציית תקשורת תומכת וחלופית (AAC) עברית-first, המיועדת
לילדים ולמבוגרים עם מוגבלויות דיבור. הנגשת המוצר לכלל המשתמשים, ובכללם
אנשים עם מוגבלות, היא ערך ליבה של המוצר ולא דרישת רגולציה בלבד.

אנו פועלים להתאים את האפליקציה להוראות תקנות שוויון זכויות לאנשים עם מוגבלות
(התאמות נגישות לשירות), התשע"ג-2013, ולתקן הישראלי ת"י 5568.

## 2. היקף ההצהרה

הצהרה זו חלה על אפליקציית הרשת (PWA) של Co_Board, על כלל המסכים והזרימות
העיקריות: לוח התקשורת, שורת המשפט וההקראה, פאנל ההגדרות, ההצטרפות (onboarding)
וניהול המנוי.

## 3. רמת ההתאמה

האפליקציה **עומדת באופן חלקי** בדרישות ת"י 5568 (WCAG 2.0 AA).
מרבית קריטריוני ההצלחה מתקיימים, לרבות:

- מבנה ARIA תקין ללוח (grid → row → gridcell) עם ניווט מקלדת.
- יעדי מגע גדולים (≥ 44px, ניתן להגדלה).
- תמיכה בסריקת מתגים (switch scanning) עם השהיית-רעד (debounce).
- גופן קריא ידידותי-דיסלקציה ומצב רגיעה חושית.
- הקראה קולית (TTS) עברית, כולל אישור לפני ניקוי וריטט (haptics) אופציונליים.
- עבודה offline-first; אין בקשות רשת לצד-שלישי לפני הסכמה.

## 4. מגבלות ידועות

נכון למועד עדכון מסמך זה, קיים **אי-עמידה יחיד ומתועד** בקריטריון:

- **WCAG 1.4.3 Contrast (Minimum):** צבע-המותג הראשי (`--cl-primary`, כתום)
  אינו מגיע ליחס ניגודיות 4.5:1 מול טקסט לבן בחלק מהכפתורים והתגים.
  מדובר בהחלטת מיתוג רוחבית ולא בבאג נקודתי. ההיבט מנוטר במפורש בחבילת
  בדיקות הנגישות (‏`app/e2e/settings-a11y.spec.ts`, כלל `color-contrast`)
  וממתין להחלטת מוצר/עיצוב. משתמשים הזקוקים לניגודיות מוגברת יכולים להפעיל
  **"ניגודיות גבוהה"** ו/או **"מצב לילה"** בהגדרות הגישה.

מגבלה זו היא הסיבה לכך שההצהרה מנוסחת כ"התאמה חלקית" ולא "התאמה מלאה".
